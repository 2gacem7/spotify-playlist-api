# Candorem Node/React Boilerplate

### To Use

1. fork the repo
2. `npm install`
3. edit .env file with environment variables you need
    (create a .env with     
    REACT_APP_SPOTIFY_CLIENT_ID=b49e02e2...bccfd42127092
    REACT_APP_SPOTIFY_CLIENT_SECRET=d4485...8e3005f1b4e39a7)
4. `npm start`
5. go to ./client
6. `npm install`
7. `npm start`
